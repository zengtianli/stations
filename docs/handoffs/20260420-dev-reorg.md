## 踩过的坑

### 🔴 mv 后 51 个 symlink 断裂

**现象**：`/handoff` `/retro` 不可用 / 终端 starship 不启动 / Raycast 失效
**根因**：`~/.claude/*` `~/.zshrc` `~/.config/*` 等 symlink 硬编码 `~/Dev/<tool>/` 旧路径
**解法**：python 脚本扫 ~/ ~/.claude/ ~/.config/ 下 symlink，相对新位置 rebuild
**固化**：`station-promote.sh` 扫描范围已扩到 `~/.claude/` `~/.config/` `~/` 顶层 dotfiles + `~/Library/LaunchAgents/`

### 🔴 BSD sed 不认 `\b` word boundary

**现象**：第一版 sed `s|~/Dev/X\b|...|` 静默失败
**根因**：macOS BSD sed 不支持 GNU `\b`
**解法**：python `re.sub(r"(?![A-Za-z0-9_-])")` 负向零宽断言

### 🔴 bash `${var/p/r}` 参数扩展保留反斜杠

**现象**：`${old/\/Dev\/X\//\/Dev\/tools\/X\/}` 替换后含字面 `\/`
**根因**：bash 不需要转义 `/`，加了就原样保留
**解法**：路径替换一律用 python

### 🟡 Python `Path.home() / "Dev" / "X"` 构造不被字符串 sed 命中

**解法**：加专用正则 `Path\.home\(\)\s*/\s*"Dev"\s*/\s*"X"` → `... / "Dev" / "tools" / "X"`

### 🟡 web-stack 移位后 .venv shebang 失效

**解法**：`rm -rf .venv && uv sync --all-packages`
