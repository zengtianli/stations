# 情绪效用冲突模型

---

| 符号       | 定义                                                      | 在本案例中的解读                                                 |
| :------- | :------------------------------------------------------ | :------------------------------------------------------- |
| **A, B** | 参与者                                                     | A = 儿子, B = 母亲                                           |
| **R**    | **角色框架 (Role Frame)**<br>个体对自身身份和关系的根本性、深层期望。           | **R_A**: "我是一个独立、平等的成年人。"<br>**R_B**: "我是一个终身、权威的母亲。"    |
| **X**    | **行为/沟通 (Action)**<br>一方所做出的具体言行。                       | **X_A**: A的言行 (如: "挺麻烦的")<br>**X_B**: B的言行 (如: "你从小就粗心") |
| **I(X)** | **解读 (Interpretation)**<br>一方如何通过自己的角色框架（R）来理解对方的行为（X）。 | **I_A(X_B)**: A对B行为的解读。<br>**I_B(X_A)**: B对A行为的解读。       |
| **E**    | **情绪效用 (Emotional Utility)**<br>互动后，个体的情绪收益（正）或损失（负）。   | **E_A, E_B**                                             |

## 公式应用分析

### 三大核心冲突分析表

| 冲突类型 | 角色框架 (R) | 行为 (X) | 解读与效用 (I & E) |
|---------|-------------|----------|-------------------|
| **1. 亲情边界与价值冲突** | **A**: "亲情是平等的，我的付出需要被看见和尊重。"<br>**B**: "亲情是无条件的，是儿子的神圣义务。" | **A**: "我无偿服务，也挺麻烦的。"<br>**B**: "你是为了你妈。" / "给你钱。" | **A行为**: B解读为"算账" → E_B < 0；A自认为主张平等 → E_A > 0<br>**B行为**: A解读为"道德绑架"和"人格侮辱" → E_A < 0；B自认为捍卫亲情神圣性 → E_B > 0 |
| **2. 沟通模式与需求冲突** | **A**: "我是解决问题的专家，需要的是信任。"<br>**B**: "我是需要被安抚的对象，需要的是共情。" | **A**: "邮编影响不大，相信我。"<br>**B**: 反复纠结和确认 | **A行为**: B解读为"敷衍和冷漠" → E_B < 0；A自认为高效解决问题 → E_A > 0<br>**B行为**: A解读为"无理取闹"和"不信任" → E_A < 0；B自认为表达真实焦虑 → E_B > 0 |
| **3. 个人成长与固化标签冲突** | **A**: "我是一个动态发展的、现在的我。"<br>**B**: "你是我所定义的、静态不变的过去的小孩。" | **A**: 解释具体原因<br>**B**: "你从小就粗心。" | **A行为**: B解读为"为粗心找借口" → E_B < 0；A自认为进行成年人的合理辩护 → E_A > 0<br>**B行为**: A解读为"成长被全盘否定" → E_A < 0；B自认为点明问题本质 → E_B > 0 |
