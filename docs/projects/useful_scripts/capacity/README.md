# 水环境功能区纳污能力计算工具

三种使用方式，业务逻辑相同，表现形式不同。

---

## 快速使用

### 方式 1：Excel VBA（推荐）

直接在 Excel 中点击按钮计算，无需 Python 环境。

1. 打开 `河道功能区输入.xlsm`
2. 按提示填写数据
3. 点击按钮自动计算

详见下方「Excel VBA 版本」。

### 方式 2：Web 界面

```bash
cd ~/useful_scripts/execute/hydraulic/capacity
streamlit run web_app.py
```

浏览器访问 http://localhost:8501

### 方式 3：命令行

```bash
cd ~/useful_scripts/execute/hydraulic/capacity

# 运行计算
python cli_run.py

# 创建示例数据
python cli_run.py --init
```

## 计算公式

### 河道纳污能力

```
W = 31.536 × b × (Cs - C0 × exp(-KL/u)) × (Q×K×L/u) / (1 - exp(-KL/u))
```

- W：纳污能力 (t/a)
- b：不均匀系数
- Cs：目标浓度（最大可接受浓度）(mg/L)
- C0：初始浓度（上断面来水浓度）(mg/L)
- K：衰减系数 (1/s)
- L：河段长度 (m)
- Q：流量 (m³/s)
- u：流速 (m/s)

### 流速公式

```
u = a × Q^β
```

### 出流浓度

```
C_out = C0 × exp(-K×L/u)
```

### 流量/浓度混合（支流汇入）

```
Q_mix = Q1 + Q2
C_mix = (Q1×C1 + Q2×C2) / (Q1 + Q2)
```

### 水库纳污能力

```
W = 31.536 × K × V × Cs × b
```

- V：库容 (m³)

## Excel VBA 版本

### 按钮清单

| 按钮 | 功能 |
|------|------|
| 初始化输入表 | 创建空白输入表结构 |
| 生成功能区列 | 根据数量生成功能区编号列 |
| 生成干支流名 | 根据支流数量生成干流/支流名 |
| 生成基础信息 | 生成基础信息 Sheet |
| 生成逐日流量 | 生成各方案的逐日流量 Sheet |
| 开始计算 | 执行计算，生成结果 |
| 清除输入 | 清空所有数据 |

### 使用流程

1. 点击「初始化输入表」
2. 填写 B1=功能区数量, B2=方案个数
3. 点击「生成功能区列」
4. 填写各功能区参数（Cs/K/b/a/β/干流长度/干流C0/支流数量）
5. 点击「生成干支流名」
6. 填写支流参数（长度/汇入位置/C0）
7. 点击「生成基础信息」
8. 点击「生成逐日流量」
9. 填写逐日流量数据
10. 点击「开始计算」

### 输出结果

每个方案生成两个 Sheet：

| Sheet 名 | 内容 | 用途 |
|----------|------|------|
| 纳污能力过程（方案N） | 干流各段 + 支流 + 混合点 + 汇总 | 核验计算过程 |
| 纳污能力结果（方案N） | 只有干流各段 + 汇总 | 提交报告 |

### 颜色说明

- 淡黄色背景：程序自动生成
- 浅黄色背景：需要手动填写

结果 Sheet：
- 白色：干流段
- 浅绿色：支流（仅过程版）
- 浅黄色：混合点（仅过程版）
- 浅蓝色 + 加粗：汇总行

## 支流分段计算说明

当功能区有支流时，干流被支流汇入点分成多段：

```
干流起点 ──段1── 支流1汇入 ──段2── 支流2汇入 ──段3── 干流终点
                    ↑                    ↑
                  支流1                支流2
```

计算流程：
1. 计算段1纳污能力，得出段1出口浓度
2. 计算支流1纳污能力，得出支流1出口浓度
3. 在汇入点进行流量/浓度混合
4. 计算段2纳污能力（使用混合后的流量和浓度）
5. 依此类推...

汇总规则：只汇总干流段的纳污能力，支流仅展示不计入汇总。
