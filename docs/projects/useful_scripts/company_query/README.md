## ⚡ 5分钟快速上手

### 第1步：安装依赖（1分钟）

打开终端，进入项目目录，运行：

```bash
pip install pandas requests openpyxl
```

或者：
```bash
pip install -r requirements.txt
```

---

### 第2步：获取天眼查API密钥（需要注册）

#### 🔸 方式1：天眼查官网（推荐）

1. 访问：**https://open.tianyancha.com/**
2. 注册账号 → 实名认证
3. 控制台 → 应用管理 → 创建应用
4. 复制 **API Token**（类似：`abc123def456`）

#### 🔸 方式2：阿里云市场（备选）

1. 访问：**https://market.aliyun.com/**
2. 搜索"企业信息查询"或"天眼查"
3. 购买API服务（通常有免费试用）

#### 🔸 方式3：暂时跳过（仅用智能识别）

如果暂时没有API密钥：
- 可以先跳过，工具仍可识别政府机关、学校、医院
- 但无法查询企业工商信息

---

### 第3步：配置API密钥（1分钟）

打开 **`config.py`** 文件，找到第8行：

```python
TIANYANCHA_API_TOKEN = "your_api_token_here"
```

替换为您的API Token：

```python
TIANYANCHA_API_TOKEN = "abc123def456ghi789"  # 替换为您的实际密钥
```

保存文件。

## ⚙️ 配置说明

### `config.py` 配置项

```python
# API配置
TIANYANCHA_API_TOKEN = "your_api_token_here"  # 您的API密钥
REQUEST_INTERVAL = 1.0                         # 请求间隔（秒）
MAX_RETRY_TIMES = 3                            # 失败重试次数
REQUEST_TIMEOUT = 10                           # 超时时间（秒）

# 文件路径
INPUT_CSV = "发曾田力-企业名称_Sheet1.csv"     # 输入文件
OUTPUT_EXCEL = "企业信息查询结果.xlsx"         # 输出文件
```

### 智能识别规则

工具会自动识别以下类型的单位（无需调用API）：

**政府机关：**
- 关键词：公安局、派出所、交警、看守所、政府、城管等

**教育机构：**
- 关键词：学校、中学、小学、幼儿园、大学、学院等

**医疗机构：**
- 关键词：医院、卫生院、诊所、中医院等

可以在 `config.py` 中自定义关键词。

## 💡 常见问题

### Q1: 提示"未配置API Token"怎么办？

**A:** 打开 `config.py`，将 `TIANYANCHA_API_TOKEN` 改为您的实际密钥。

如果暂时没有API密钥，工具仍会使用智能识别功能，但无法查询企业工商信息。

### Q2: 出现"查询失败"或"API返回错误"

**可能原因：**
- API密钥无效或过期
- API调用次数用完
- 企业名称不存在
- 网络连接问题

**解决方法：**
1. 检查API密钥是否正确
2. 登录天眼查平台查看剩余次数
3. 检查网络连接

### Q3: 如何查看API剩余次数？

登录天眼查开放平台：https://open.tianyancha.com/  
进入"控制台" → "用量统计"

### Q4: 为什么有些企业查询失败？

**常见原因：**
- 政府机关、事业单位没有工商注册信息（会自动智能识别）
- 企业名称拼写错误
- 企业已注销
- 非规范企业名称

### Q5: 如何调整查询速度？

修改 `config.py` 中的 `REQUEST_INTERVAL`：
```python
REQUEST_INTERVAL = 0.5  # 更快（可能被限流）
REQUEST_INTERVAL = 2.0  # 更慢（更稳定）
```

### Q6: API费用是多少？

**天眼查开放平台：**
- 企业基本信息查询：约0.5-1元/次
- 通常有100-500次免费额度

**阿里云市场：**
- 不同供应商价格不同
- 部分提供包年套餐

### Q7: 可以使用其他API吗？

可以！支持：
- 企查查API
- 启信宝API
- 其他企业信息API

需要修改 `company_industry_query.py` 中的API调用部分。

## 🔧 进阶使用

### 自定义智能识别规则

编辑 `config.py`，添加自定义关键词：

```python
INSTITUTION_INDUSTRY_MAP = {
    "政府机关": ["公安局", "派出所", "你的关键词"],
    "教育": ["学校", "中学", "你的关键词"],
    "医疗卫生": ["医院", "诊所", "你的关键词"],
    "新增行业": ["关键词1", "关键词2"]  # 添加新行业
}
```

### 批量处理多个文件

修改 `company_industry_query.py` 的 `main()` 函数：

```python
def main():
    query = CompanyIndustryQuery()
    
    files = [
        ("文件1.csv", "结果1.xlsx"),
        ("文件2.csv", "结果2.xlsx"),
    ]
    
    for input_file, output_file in files:
        query.process_csv(input_file, output_file)
```
