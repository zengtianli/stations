# 绍兴市域水资源承载能力评价

## 项目背景

绍兴市经济快速发展，水资源需求持续增长，面临以下问题：

- **供需矛盾**：经济发展与水资源供给之间的矛盾日益突出
- **评价缺失**：缺乏科学的水资源承载能力评价体系
- **决策支撑不足**：水资源规划缺乏量化依据

项目旨在构建科学的评价体系，为水资源规划提供决策支撑。

## 技术方案

### 指标体系构建

构建多维度评价指标体系：

| 维度 | 指标示例 |
|------|----------|
| 水量 | 人均水资源量、供水保证率 |
| 水质 | 水功能区达标率、饮用水源水质 |
| 生态 | 生态用水保障率、河湖健康指数 |
| 效率 | 万元GDP用水量、灌溉水利用系数 |

### 评价模型开发

采用多准则决策方法：

1. **AHP（层次分析法）**：确定指标主观权重
2. **CRITIC（客观赋权法）**：确定指标客观权重
3. **组合赋权**：综合主客观权重
4. **TOPSIS（逼近理想解法）**：计算综合评价值

### 软件系统开发

开发 GUI 软件系统：

- 数据导入与管理
- 权重计算模块
- 评价计算模块
- 结果可视化与导出

## 我的贡献

2. **模型开发**：开发 AHP + CRITIC + TOPSIS 综合评价模型
3. **软件开发**：开发水资源承载力评价 GUI 软件系统
4. **成果推广**：评价方法被省水利厅采纳推广
