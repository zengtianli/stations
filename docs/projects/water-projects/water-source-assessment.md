# 仙居县饮用水水源地安全保障达标评估

## 项目背景

饮用水水源地安全保障达标评估是保障城乡居民饮水安全的重要制度。仙居县西岙水库、盂溪水库作为县域重要饮用水水源地，需每年开展安全保障达标评估：

- **评估范围**：台州市仙居县西岙水库、盂溪水库
- **评估周期**：年度评估（2023-2025连续3年）
- **评估要求**：涵盖水量保障、水质保障、监控能力、管理体系等多维度

## 技术方案

### 评估指标体系

完成20大类评估指标的材料编制：

| 类别 | 评估内容 |
|------|----------|
| 供水保障 | 供水保障程度、来水量保证率计算 |
| 应急备用 | 应急备用水源建设情况 |
| 水量调度 | 水量调度管理、供水安全系数分析 |
| 设施运行 | 取输水设施运行状况 |
| 水质监测 | 取水口水质、水质达标率统计 |
| 保护区管理 | 保护区划分与综合治理 |
| 监测预警 | 监测预警体系建设 |
| 应急保障 | 应急能力保障 |
| 信息化 | 信息化管理水平 |
| 资金保障 | 资金投入与保障 |

### 技术指标计算

- 来水量保证率计算
- 供水安全系数分析
- 水质达标率统计

## 我的贡献

1. **项目负责人**：县局层面全面负责，统筹评估全流程
2. **材料编制**：完成20大类评估指标材料编制
3. **技术分析**：完成来水量保证率、供水安全系数、水质达标率等技术指标计算
4. **支撑材料**：整理应急演练记录、部门联动台账等200余份支撑材料
5. **多部门协调**：协调仙居县水利局、台州市生态环境局仙居分局、盂溪水库发展有限公司、永安水务集团等4个单位完成材料审核与盖章确认

## 项目成果

| 成果 | 说明 |
|------|------|
| 评估结果 | 连续3年（2023-2025）评估结果优秀 |
| 覆盖水源地 | 西岙水库、盂溪水库 |
| 支撑材料 | 200余份评估证明材料 |
| 协调单位 | 4个部门/单位 |
