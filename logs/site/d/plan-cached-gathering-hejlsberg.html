`lib_greeks.py:215` `net_theta = sum(r["qty"] * r["theta"] * 100 for r in opt_rows)` — 数学对 - 单张 theta 来自 Black-Scholes 公式 `lib_greeks.py:104-120`，在 `T → 0` 时 theta 分母 `sqrt(T)` → 0，数值爆炸 - 实测 `QQQ 260421C00615000` (DTE=0) 单张 theta = −6.80，`qty * theta * 100 = +1359`/day，占 net_theta 的 55% - 另有 `QQQ 260417C00568000` DTE=**−4**，Robinhood 数据同步滞后还留在 portfolio 里 **"有效 Theta" 定义**： ```python effective_theta = sum(r["qty"] * r["theta"] * 100 for r in opt_rows if r["dte"] >= 1) # 当前真实值 ≈ $1,094/day（排掉 2 张问题合约后） ``` **展示规则**：UI 只显示"有效 Theta"一个数字；每张合约明细表里，DTE≤0 的行**标灰 + 标注 "已过期/今日到期"** 但不计入总和。 --- ## §1 · Tab 1 总览 · 要素清单 ### 1.1 StatCards 组 A（4 格，资产结构） | 字段 | 公式 | 来源 | |---|---|---| | NLV | `data['nlv']` | process_portfolio | | QQQ 持仓市值 | `data['stock_mv']` | process_portfolio | | Margin 借款 | `data['margin_debt']` | process_portfolio | | 杠杆倍数 | `data['leverage']` 格式 `{x:.1f}×` | process_portfolio | ### 1.2 资产结构解释表（4 行） 列：**指标 · 数值 · 怎么算的** -
