# Hammerspoon 整合 Raycast 功能 + Web Dashboard ## Context Hammerspoon 和 Raycast 有 8 个功能完全重复（如 file_run、app_open、file_copy），导致用户找不到命令（以为在 Raycast 实际在 HS）。用户希望： 1. 把适合 hotkey 驱动的 Raycast 功能迁移到 Hammerspoon 2. 清理重复命令 3. 建一个 Web Dashboard 随时查看 HS 快捷键配置 ## 命令处置总表 ### 删除：8 个纯重复 Raycast 命令 | Raycast 命令 | HS 已有 | 快捷键 | |---|---|---| | `file_run.sh` | `apps.run_script()` | ⌘⌃⇧S | | `file_run_parallel.sh` | `apps.run_scripts_parallel()` | ⌘⌃⇧R | | `app_open.sh` | `open_terminal/ide/nvim` | ⌘⌃⇧T/W/I | | `file_copy.sh` | `copy_filenames/content` | ⌘⌃⇧N/B | | `folder_paste.sh` | `paste_to_finder` | ⌃⌥V | 要删的 wrapper 文件： - `devtools/raycast/commands/file_run.sh` - `devtools/raycast/commands/file_run_parallel.sh` - `devtools/raycast/commands/app_open.sh` - `mactools/raycast/commands/file_copy.sh` - `mactools/raycast/commands/folder_paste.sh` ### 迁移到 HS：13 个命令（3 组） **组 1：ClashX 代理（6 命令）→ 新建 `modules/network.lua`** 参考实现：`~/Dev/clashx/lib/clashx.sh`，全部可用 `hs.osascript` + `hs.http` + `hs.execute` 原生替代。 | 功能 | 快捷键 | 实现方式 | |---|---|---| | Toggle proxy | Hyper+1 | `hs.osascript('tell app "ClashX Pro" to toggleProxy')` | | Mode: Rule | Hyper+2 | `hs.osascript('tell app "ClashX Pro" to proxyMode "rule"')` | | Mode: Global | Hyper+3 | 同上 `"global"` | | Mode: Direct | Hyper+4 | 同上 `"direct"` | | Toggle Enhanced | Hyper+5 | System Events GUI 自动化 | | Show Status | Hyper+6 | `hs.http.get` + `networksetup` 组合 → `hs.alert` | 额外增强：`hs.menubar` 显示当前代理模式图标（Raycast 做不到）。 **组 2：Yabai 窗口管理（5 命令）→ 新建 `modules/window.lua`** 参考实现：`~/Dev/mactools/scripts/window/yabai.py`，每个函数就是一行 `yabai -m` 命令。 | 功能 | 快捷键 | 实现 | |---|---|---| | Toggle float | Hyper+F | `hs.execute("yabai -m window --toggle float")` | | Mouse follow | Hyper+M | 读当前值 → toggle | | Organize | Hyper+O | bsp → balance → float 三步 | | Toggle yabai | Hyper+Y | pgrep → start/stop-service | | Restart yabai | Hyper+⇧Y | restart-service | **组 3：显示器 + 系统（5 命令）→ 扩展现有模块** | 功能 | 快捷键 | 目标模块 | 实现 | |---|---|---|---| | Display 1080p | Hyper+7 | system | `hs.task` 调用 `displayplacer`，逻辑从 `display_1080.sh` 移植 | | Display 4K | Hyper+8 | system | 同上，分辨率参数不同 | | Brew maintain | Hyper+B | system | 在终端里执行（长时间任务） | | App launcher | Hyper+A | apps | `hs.task` 调用 `sys_app_launcher.py` | | File print | Hyper+P | system (finder-scoped) | `hs.task` + `lpr` | ### 保留在 Raycast：~42 个命令 - 文档处理 17 个（python-docx 重度依赖、进度输出） - 数据转换 13 个（openpyxl/pandas） - 水利 Streamlit 5 个（启动 Web UI） - 复杂开发工具 4 个（git-smart-push、tts、cc-export、downloads-cleanup） - cclog 3 个（数据处理重） 这些不适合 hotkey 模式，Raycast 的 fullOutput + 参数 UI 更合适。 ## Dashboard 方案 ### 架构 匹配现有 `repo-dashboard` 模式：Streamlit 单页 → VPS 部署 → 子域名 + CF Access。 ``` ~/.hammerspoon/lib/export_config.lua ← 新增：读 hotkeys.lua 导出 JSON ~/Dev/hs-dashboard/ ← 新建项目 ├── app.py # Streamlit 单页 ├── data/hs_config.json # HS 配置 JSON（export_config.lua 生成） ├── deploy.sh # rsync + systemctl restart └── requirements.txt # streamlit ``` **数据流**： 1. `export_config.lua` 在 HS reload 时自动导出 `hs_config.json` 2. `deploy.sh` 先调 `hs -c "require('lib.export_config').export()"` 确保数据最新 3. rsync → VPS → systemctl restart **JSON 内容**：hotkeys 表（快捷键/描述/模块/作用域）、模块列表、设置项 **VPS 部署**： - 域名：`hammerspoon.tianlizeng.cloud` - 端口：852X（同 dashboard 系列） - systemd：`hs-dashboard` - CF DNS + CF Access ## 实施顺序 ### Phase 1：删除重复 Raycast 命令 - 删 5 个 wrapper 文件 - `raycast sync.py --apply` 清理 - 63 → 58 个命令 ### Phase 2：迁移 ClashX → `modules/network.lua` - 新建模块，6 个函数 + menubar 指示器 - hotkeys.lua 加 6 行 - 验证后删 clashx raycast wrappers ### Phase 3：迁移 Yabai + Display → `modules/window.lua` + `system.lua` - 新建 window 模块 5 个函数 - system 加 display 切换 2 个函数 - hotkeys.lua 加 7 行 ### Phase 4：迁移 Tier 2 系统命令 - system/apps 模块各加 2-3 个函数 - hotkeys.lua 加 3 行（brew/app-launcher/print） ### Phase 5：Dashboard - 新建 `export_config.lua` + `~/Dev/hs-dashboard/` - VPS 部署 - 域名配置 ### Phase 6：清理收尾 - 删已迁移的 Raycast wrappers（clashx 6 + yabai 5 + display 2 + brew/app-launcher/print 3 = 16） - 最终 Raycast：63 - 8(重复) - 13(迁移) ≈ 42 个命令 - 更新各 repo CLAUDE.md ## 关键文件 | 文件 | 操作 | |---|---| | `~/.hammerspoon/lib/hotkeys.lua` | 新增 ~16 行快捷键声明 | | `~/.hammerspoon/modules/network.lua` | 新建：ClashX 6 函数 + menubar | | `~/.hammerspoon/modules/window.lua` | 新建：Yabai 5 函数 | | `~/.hammerspoon/modules/system.lua` | 扩展：display + brew + print | | `~/.hammerspoon/modules/apps.lua` | 扩展：app-launcher | | `~/.hammerspoon/lib/export_config.lua` | 新建：JSON 导出 | | `~/Dev/hs-dashboard/app.py` | 新建：Streamlit dashboard | | `~/Dev/hs-dashboard/deploy.sh` | 新建：部署脚本 | | `~/Dev/clashx/lib/clashx.sh` | 参考（迁移源） | | `~/Dev/mactools/scripts/window/yabai.py` | 参考（迁移源） | ## 验证 -
