# 开发工具

我的开源开发工具集合，涵盖了 macOS 自动化、编辑器配置、命令行环境等各个方面，旨在提升开发效率和体验。

## 工具概览

这里收录了我在开发过程中打磨的各种工具和配置，每个工具都经过了长期的使用和优化，具有以下特点：

- **实用性强**：解决实际开发中的痛点问题
- **高度优化**：注重性能和用户体验
- **模块化设计**：易于扩展和维护
- **完整文档**：详细的安装和使用说明
- **开源共享**：MIT 协议，欢迎贡献和反馈

## 技术栈

涵盖的技术栈包括：Shell、Python、Lua、TypeScript、AppleScript、Yabai、Raycast、Neovim、Zsh 等。

所有工具都在 GitHub 和 Gitee 上开源，欢迎 Star、Fork 和贡献代码！
